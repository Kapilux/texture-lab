# 🎨 Image Texture & Noise Generator (Free Online Tool)

A free and easy-to-use online tool to add **patterns, noise, and texture effects** to images. Create unique visuals with customizable overlays like grain, dots, lines, waves, and hexagons — perfect for designers, content creators, and developers.

---

## 🚀 Features

* Add texture overlays:

  * Noise / Grain
  * Crosshatch
  * Dots
  * Lines
  * Waves
  * Hexagon patterns
  * Circuit

* Fully customizable:

  * Adjust intensity
  * Change pattern size
  * Rotate angles
  * Zoom control

* Apply color styles:

  * Noise
  * Grayscale filter
  * Brightness
  * Contrast
  * Custom color overlay

* Works directly in the browser (no installation required)
* Export PNG

---

## 🖼️ Use Cases

* Enhance social media graphics
* Create retro or vintage-style images
* Add grain/noise for aesthetic photography
* Generate textures for UI/UX design
* Backgrounds for websites or presentations
